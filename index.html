About Me

Saya adalah fresh graduate dari kampus iSTTS pada tahun 2023, dimana saya menekuni passion saya di bidang teknologi dengan mengambil jurusan D3 Sistem Informasi. Saya cukup mengikuti perkembangan teknologi, terutama komputer dan internet. Sebagai web developer, saya menguasai beberapa bahasa pemrograman seperti PHP, HTML, JavaScript, jQuery, dan menggunakan Framework Laravel yang merupakan beberapa keahlian saya. Dan dengan beberapa keahlian dalam dan implementasi API seperti Midtrans, saya dapat membuat situs web yang interaktif dan dinamis dengan integrasi pembayaran untuk mempermudah pengalaman pengguna. Proyek magang dalam membuat website warehouse untuk "CV Tekad Maju Jaya" pada tahun 2021 adalah pengalaman berharga yang memberikan wawasan dalam industri yang berharga.

Selain kemampuan saya dalam software komputer, saya juga mendalami bidang hardware komputer. Mempelajari cara maintenance dan merakit komputer telah memberi saya pemahaman yang lebih dalam tentang teknologi yang saya gunakan setiap hari. Saya juga memiliki sisi kreatif yang mengkspresikan diri melalui seni visual, saya pernah bergabung dengan organisasi turnamen game e-sports yaitu "Bridge Esports Indonesia" (sebelumnya bernama "Indonesia Vainglory League") pada tahun 2018 memungkinkan saya untuk mengembangkan bakat dalam mengedit logo dan membuat banner atau poster untuk bahan promosi. Saya juga tertarik pada bidang fotografi membuat saya mampu mengoperasikan sebuah kamera, mengambil beberapa gambar yang saya posting di akun Instagram saya.

Projects

Website Tournament

Website untuk membuat dan mengikuti turnamen, website ini dibuat dengan Laravel Framework dan dengan MySQL database, menyediakan fitur untuk pendaftaran turnamen dan pembayaran menggunakan Midtrans sandbox, dan dapat membuat turnamen dengan single dan double elimination.

Website Warehousing

Website untuk pencatatan dan pengelolaan spare part mesin di gudang, website ini dibuat dengan menggunakan bahasa pemrograman PHP dan MySQL database, menyediakan fitur untuk melihat semua catatan karena setiap operator gudang dapat mencatat kapan part mesin bertambah, berkurang atau dipindahkan.

Photography

Beberapa gambar yang saya ambil menggunakan kamera Fujifilm S2980 dan Canon EOS 50D, agar hasil gambar terlihat dramatis atau realistis saya menggunakan software Adobe Lightroom, terkadang saya juga menggunakan Adobe Photoshop untuk membuat beberapa efek tambahan.

Review Hardware

Saya lebih sering membuat review untuk cooler untuk komputer dan membandingkan performa sebelum dan sesudah menggunakan cooler tertentu yang saya edit menggunakan Adobe Premiere Pro, dan terkadang jika video tersebut sangat singkat saya menggunakan CapCut.

Making Logo

Logo yang saya buat dengan Adobe Photoshop and Pixellab untuk Bridge Esports Indonesia yang merupakan organisasi turnamen esports game Vainglory (sebelumnya bernama Indonesia Vainglory League). Selain logo, saya juga membuat beberapa poster untuk promosi.
